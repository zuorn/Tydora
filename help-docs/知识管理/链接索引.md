# 链接索引

链接索引是 Tydora 的后台服务，负责构建和维护笔记间的链接关系。

## 功能说明

- 自动扫描仓库中的所有笔记
- 构建出链（outlinks）索引
- 构建反链（backlinks）索引
- 构建文件名索引
- 支持增量更新

## 工作机制

### 索引构建

1. 启动时扫描整个仓库
2. 解析所有 Markdown 文件中的 [[Wiki链接]]
3. 构建双向链接关系表

### 增量更新

- 监听文件系统变化
- 仅更新修改的文件的链接关系
- 支持文件创建、删除、重命名

### 索引内容

| 索引类型 | 说明 |
|----------|------|
| 出链 | 当前笔记链接了哪些笔记 |
| 反链 | 哪些笔记链接了当前笔记 |
| 文件名 | 文件名到路径的映射 |

## 性能优化

- 批量处理（每次 50 个文件）
- 后台异步构建
- 不阻塞编辑器操作

## 使用场景

- [[反向链接]] 面板显示
- [[知识图谱]] 渲染
- [[Wiki链接]] 自动补全
- [[快速打开]] 文件搜索
