# 文件树

侧栏文件树显示当前仓库的目录结构。

## 功能说明

- 递归显示目录和文件
- 目录默认折叠，点击展开
- 目录始终排在文件前面
- 按文件名排序

## 操作

### 展开/折叠

- 点击目录前的箭头展开或折叠
- 双击目录名也可展开/折叠

### 右键菜单

在文件或目录上右键点击，可执行：

- **新建文件** — 在当前目录创建新文件
- **新建文件夹** — 在当前目录创建新文件夹
- **重命名** — 内联重命名
- **删除** — 删除文件或文件夹（需确认）
- **复制路径** — 复制文件的完整路径

### 拖拽移动

1. 按住鼠标左键拖拽文件
2. 拖拽到目标目录
3. 释放鼠标完成移动

### 内联重命名

1. 右键选择"重命名"，或按 `F2`
2. 文件名变为可编辑状态
3. 输入新名称，按 `Enter` 确认
