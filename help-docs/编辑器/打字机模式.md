# 打字机模式

打字机模式将光标始终保持在屏幕中央，提供更专注的写作体验。

## 功能说明

启用打字机模式后，当光标移动到编辑区域的上半部分时，编辑器会自动滚动，使光标始终保持在屏幕中央位置。

**优势：**
- 减少视觉疲劳
- 保持专注写作
- 避免频繁滚动

## 启用方法

1. 按 `Ctrl+Shift+T` 快速切换打字机模式
2. 或在 [[设置/编辑器设置]] 中启用"打字机模式"选项

## 使用场景

- 长文写作
- 需要专注的写作任务
- 习惯于中心定位的用户
