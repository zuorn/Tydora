# 快速打开

快速打开功能让你快速查找并打开仓库中的文件。

## 使用方法

1. 按 `Ctrl+O` 打开快速打开面板
2. 输入文件名关键词进行模糊搜索
3. 从搜索结果中选择文件
4. 按 `Enter` 打开文件

## 模糊搜索

快速打开支持模糊搜索，无需输入完整文件名：

- 输入 `笔记` 可匹配 `我的笔记.md`、`笔记管理.md` 等
- 输入 `2024日` 可匹配 `2024年日记.md`
- 输入 `项目 readme` 可匹配 `项目/README.md`

## 最近文件

快速打开面板会显示最近打开的文件，方便快速访问常用文件。

## 搜索范围

搜索范围为当前活跃 [[仓库]] 的所有文件。
