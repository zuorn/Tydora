# 关于 Tydora

Tydora 是一款基于 Tauri 构建的现代化桌面 Markdown 编辑器。

## 版本信息

- 当前版本：0.1.0
- 构建框架：Tauri v2
- 前端框架：React 19
- 编辑器引擎：Vditor 3.x

## 检查更新

1. 打开设置（`Ctrl+,`）
2. 切换到"关于"标签
3. 点击"检查更新"

Tydora 支持自动更新，启动时会自动检查是否有新版本。

## 功能特性

- 三种编辑模式：WYSIWYG、即时渲染、源码
- 多仓库管理
- Wiki 双链笔记
- 知识图谱可视化
- 思维导图
- 8 个内置主题
- 40+ 可自定义快捷键
- 图片、视频、音频、PDF 预览

## 技术栈

- **后端**：Rust (Tauri v2)
- **前端**：React 19 + TypeScript + Vite 6
- **编辑器**：Vditor 3.x
- **图表**：D3.js、Markmap

## 开源许可
