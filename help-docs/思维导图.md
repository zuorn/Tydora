# 思维导图

Tydora 内置思维导图功能，基于 Markmap 库，可以将 Markdown 标题转换为可交互的思维导图。

## 使用方法

1. 按 `Ctrl+M` 打开思维导图
2. 在独立窗口中查看思维导图
3. 使用鼠标滚轮缩放
4. 拖拽画布调整视图
5. 点击节点可折叠/展开

## 功能特性

### 缩放与平放

- 滚轮缩放
- 拖拽平移
- 双击居中

### 折叠与展开

- 点击节点可折叠子节点
- 点击已折叠节点可展开
- 支持全部折叠/展开

### 缩放级别设置

可在 [[设置/思维导图设置]] 中配置：
- 默认展开层级
- 动画时长
- 节点间距

## Markdown 格式

思维导图基于标题层级生成：

```markdown
# 根节点
## 一级节点 1
### 二级节点 1.1
### 二级节点 1.2
## 一级节点 2
### 二级节点 2.1
```

## 使用场景

- 大纲可视化
- 创意发散
- 项目规划
- 知识整理
