# 文件操作

Tydora 提供完整的文件管理功能。

## 新建文件

1. 在侧栏文件树中右键点击目录
2. 选择"新建文件"
3. 输入文件名（如 `笔记.md`）
4. 按 `Enter` 确认

## 新建文件夹

1. 在侧栏文件树中右键点击目录
2. 选择"新建文件夹"
3. 输入文件夹名称
4. 按 `Enter` 确认

## 重命名

### 方法一：右键菜单

1. 右键点击文件或文件夹
2. 选择"重命名"
3. 输入新名称
4. 按 `Enter` 确认

### 方法二：快捷键

1. 选中文件或文件夹
2. 按 `F2`
3. 输入新名称
4. 按 `Enter` 确认

## 删除

1. 右键点击文件或文件夹
2. 选择"删除"
3. 在确认对话框中点击"确定"

> 删除操作不可撤销，请谨慎操作。

## 拖拽移动

1. 在文件树中按住文件
2. 拖拽到目标目录
3. 释放鼠标完成移动

## 复制路径

1. 右键点击文件
2. 选择"复制路径"
3. 文件的完整路径将复制到剪贴板
