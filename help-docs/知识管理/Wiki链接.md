# Wiki 链接

Wiki 链接是 Tydora 的核心功能之一，让你轻松建立笔记间的双向链接。

## 基本语法

### 链接到笔记

```markdown
[[笔记名]]
```

### 带别名的链接

```markdown
[[笔记名|显示文本]]
```

### 链接到标题

```markdown
[[笔记名#标题]]
```

## 使用方法

1. 在编辑器中输入 `[[`
2. 开始输入笔记名
3. 从自动补全列表中选择
4. 按 `Enter` 确认

## 自动补全

输入 `[[` 后，Tydora 会自动显示匹配的笔记列表。支持模糊搜索，输入关键词即可筛选。

## 创建新笔记

如果链接的笔记不存在，点击链接会自动创建新笔记。

## 嵌入内容

使用感叹号可以嵌入其他笔记的内容：

```markdown
![[笔记名]]
```

> 详见 [[嵌入内容]]

## 反向链接

当其他笔记链接到当前笔记时，反向链接面板会显示来源。

> 详见 [[反向链接]]
