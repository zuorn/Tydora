# Tydora 帮助文档

欢迎使用 Tydora 帮助文档。Tydora 是一款基于 Tauri 构建的现代化桌面 Markdown 编辑器，支持知识管理、双链笔记、思维导图等功能。

## 《Tydora 序》

古之学者，载笔以记，竹简缣帛，各有其便。至于今日，屏上为纸，键盘为笔，然散佚无序者众矣。余乃效 Typora 之风，制此软件，名曰 Tydora——"Ty"者，Type 与 Typography 之省，文字排版之本也；"dora"者，古希腊语 δῶρον，礼物之意。合而观之，指尖之馈也。

其编辑也，三 mode 并立：所见即所得，如临纸上；即时渲染，落笔即成；源码模式，纤毫毕现。随心而易，无滞于形。

其文件管理也，立仓库以统群籍，建树形以理万端。拖拽移易，命名删除，皆在一指之间。多仓并置，各有所属，不相混淆。

其知识管理也，缀以方括之链，甲篇之论可贯乙章之旨；反向链接，使吾知所从来；知识图谱，点若列宿，线若虹霓，万象森然。嵌入他篇，一文可得众妙。

其思维导图也，以 Markmap 生成，一图统览，脉络分明，缩放折叠，随心所欲。

其主题也，八色并陈，昼夜咸宜；代码高亮，亦有十种之选。

昔庄子有言："筌者所以在鱼，得鱼而忘筌。"今 Tydora 者，亦筌蹄之属也。苟能因器通思，得意忘形，则此馈也，庶几不负矣。
