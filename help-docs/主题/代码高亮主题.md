# 代码高亮主题

Tydora 支持多种代码语法高亮主题，可独立于应用主题设置。

## 可用主题

### 浅色主题

- **Atom One Light** — Atom 编辑器浅色主题
- **GitHub Light** — GitHub 风格浅色主题
- **VS Code Light** — VS Code 默认浅色主题

### 深色主题

- **Atom One Dark** — Atom 编辑器深色主题
- **GitHub Dark** — GitHub 风格深色主题
- **VS Code Dark** — VS Code 默认深色主题
- **Nord** — 北欧风格冷色调主题
- **Monokai** — 经典的 Monokai 主题
- **Dracula** — 吸血鬼风格深色主题
- **Solarized** — Solarized 配色方案

## 切换主题

1. 打开设置（`Ctrl+,`）
2. 切换到"编辑器"标签
3. 在"代码高亮主题"下拉菜单中选择

## 自动切换

默认情况下，代码高亮主题会根据应用主题自动切换：

- 浅色应用主题 → 浅色代码主题
- 深色应用主题 → 深色代码主题

可在设置中关闭自动切换，手动选择固定主题。

## 预览效果

在 [[设置/编辑器设置]] 中可以预览各主题的代码高亮效果。
