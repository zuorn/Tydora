# 知识图谱

知识图谱以可视化方式展示笔记间的链接关系，帮你发现知识网络。

## 功能说明

- 基于 D3.js 力导向图渲染
- 节点大小反映链接数量
- 边的粗细反映链接频率
- 支持缩放和拖拽

## 使用方法

1. 按 `Ctrl+G` 打开知识图谱
2. 在独立窗口中查看笔记关系
3. 点击节点打开对应笔记
4. 使用鼠标滚轮缩放
5. 拖拽节点调整布局

## 图谱解读

### 节点

- 圆形大小表示该笔记的链接数量
- 被引用越多的笔记，节点越大
- 颜色表示笔记的分组（如果有）

### 连线

- 连线表示笔记间的 [[Wiki链接]] 关系
- 连线粗细表示链接频率
- 箭头方向表示引用方向

### 布局

- 力导向图会自动调整布局
- 密集关联的笔记会聚集在一起
- 孤立的笔记会分布在边缘

## 使用场景

- 发现知识网络的结构
- 找到核心笔记（节点最大的）
- 发现孤立笔记（无连线的）
- 理解知识体系的整体脉络
