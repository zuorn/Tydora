# 大纲面板

大纲面板显示当前文档的标题结构，方便快速导航。

## 功能说明

- 自动解析 Markdown 标题（1-6 级）
- 显示标题层级结构
- 支持点击跳转
- 显示当前阅读位置

## 使用方法

1. 在侧栏顶部切换到"大纲"标签
2. 查看当前文档的标题列表
3. 点击任意标题，编辑器滚动到对应位置

## 标题层级

大纲面板支持所有 6 级标题：

```markdown
# 一级标题
## 二级标题
### 三级标题
#### 四级标题
##### 五级标题
###### 六级标题
```

## 使用场景

- 长文档快速导航
- 查看文档结构
- 跳转到特定章节
