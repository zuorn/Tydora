# Tydora — 现代 Markdown 编辑器

## 关于命名

**Tydora** 的诞生，首先源于对 **Typora** 的致敬。我们希望在继承经典的同时，赋予它新的生命力。

命名时，我们保留“Ty”开头和“ora”结尾，韵律几乎一样，但中间换成“d”，听起来像“Tide（潮汐）”，暗示写作如流水般顺畅。这一巧思，既是对 Typora 的温情回应，也是对流畅写作体验的浪漫期许。

在此基础上，名字的每一部分还有更深层的含义：

- **“Ty”** —— 取自 **Type**（打字）与 **Typography**（文字排版），致敬文字创作的纯粹本质。
- **“dora”** —— 源自古希腊语 **δῶρον** (Dōron)，意为 **“礼物”**。

合在一起，**Tydora** 便是 **“指尖上的礼物”**。我们相信，优秀的工具不应成为思考的阻碍。Tydora 延续了 Typora“所见即所得”的极简理念，并融入更多现代创作者的期待，将沉浸式写作的体验作为一份礼物，献给每一位热爱文字的人。让每一次敲击键盘，都如行云流水般自然从容。
