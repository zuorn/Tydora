# 思维导图设置

配置 [[思维导图]] 和 [[知识图谱]] 的显示参数。

## 打开设置

按 `Ctrl+,` 打开设置面板，切换到"思维导图"标签。

## 思维导图设置

### 最大宽度

设置思维导图节点的最大宽度。

### 间距

设置节点之间的间距。

### 动画时长

设置展开/折叠动画的持续时间（毫秒）。

### 线条宽度

设置连接线的宽度。

### 颜色冻结层级

设置从第几层开始使用不同颜色。

## 知识图谱设置

### 节点大小

设置图谱节点的默认大小。

### 连线粗细

设置图谱连线的粗细。

### 力度

调整力导向图的物理模拟力度。
