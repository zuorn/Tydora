# 右键菜单

在编辑区域右键点击可打开上下文菜单，提供快速格式化和编辑命令。

## 基本操作

- **撤销** / **重做** — 回退或恢复操作
- **剪切** / **复制** / **粘贴** — 标准剪贴板操作

## 格式化

- **加粗** — `Ctrl+B`
- **斜体** — `Ctrl+I`
- **删除线** — `Ctrl+D`
- **行内代码** — 包裹选中文本为行内代码

## 标题

选择文本后，可快速设置标题级别（1-6 级）或转换为段落。

## 列表

- **无序列表** — 项目符号列表
- **有序列表** — 数字编号列表
- **任务列表** — 带复选框的列表

## 块级元素

- **引用** — 添加引用块
- **代码块** — 插入代码块
- **表格** — 插入表格
- **水平分割线** — 插入分隔线

## 插入

- **超链接** — 插入链接
- **图片** — 插入图片
- **Wiki 链接** — 插入 [[Wiki链接]]
