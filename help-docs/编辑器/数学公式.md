# 数学公式

Tydora 支持 KaTeX 和 MathJax 两种数学公式渲染引擎。

## 行内公式

使用单个美元符号包裹公式：

```markdown
$E=mc^2$
```

显示效果：$E=mc^2$

## 块级公式

使用两个美元符号包裹公式：

```markdown
$$
\sum_{i=1}^{n} i = \frac{n(n+1)}{2}
$
```

## 常用语法

### 上标与下标

```markdown
$x^2$    → 上标
$H_2O$   → 下标
$x_i^{2n}$ → 组合
```

### 分数

```markdown
$\frac{a}{b}$
```

### 根号

```markdown
$\sqrt{x}$     → 平方根
$\sqrt[n]{x}$  → n次方根
```

### 求和与积分

```markdown
$\sum_{i=1}^{n}$     → 求和
$\int_{a}^{b}$       → 积分
$\prod_{i=1}^{n}$    → 连乘
```

### 希腊字母

```markdown
$\alpha$ $\beta$ $\gamma$ $\delta$
$\pi$ $\sigma$ $\omega$ $\theta$
```

### 矩阵

```markdown
$$
\begin{bmatrix}
1 & 2 \\
3 & 4
\end{bmatrix}
$
```

## 引擎切换

可在 [[设置/编辑器设置]] 中切换公式渲染引擎：

- **KaTeX** — 渲染速度快，语法兼容性好（推荐）
- **MathJax** — 功能更全面，支持更多 LaTeX 语法
