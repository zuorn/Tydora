# 图片设置

配置图片的存储方式、命名格式等选项。

## 打开设置

按 `Ctrl+,` 打开设置面板，切换到"图片"标签。

## 存储模式

### 仓库资源（推荐）

图片保存在当前仓库的指定文件夹中。

- 优点：图片随仓库一起管理
- 缺点：仓库体积会增大

### 固定目录

图片保存在固定的本地目录中。

- 优点：所有图片集中管理
- 缺点：路径可能失效

### 图床

图片上传到第三方图床。

- 优点：不占用本地空间
- 缺点：需要网络，可能有费用

## 文件命名格式

设置粘贴/拖拽图片时的文件名格式：

- `年月日时分秒` — 如 `20240101120000.png`
- `随机字符串` — 如 `abc123.png`
- `原始文件名` — 保留原始文件名

## 固定目录路径

当选择"固定目录"存储模式时，设置图片的保存路径。

## 粘贴行为

设置粘贴图片时的默认行为：

- 自动保存到仓库
- 询问保存位置
- 仅插入引用

## 支持的格式

粘贴和拖拽支持以下格式：

- PNG
- JPG/JPEG
- GIF
- WebP
- BMP
