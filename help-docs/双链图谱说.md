# **《双链图谱说》**

余制软件曰 tydora，以 Markdown 为牒，记所览典籍，缀以方括之链，构以星罗之图。既成，叹其双链图谱之妙，然复静思：此器岂可尽学耶？

夫双链者，若经纬交织，一引则千端皆应。甲篇之论，可贯乙章之旨；前录之疑，忽解后记之惑。其反向之示，犹暮鼓晨钟，使吾知所从来。图谱观之，点若列宿，线若虹霓，或稠或疏，则吾心之所注，识之所聚，历历可辨。静览全局，则万象森然；动察一隅，则毫忽毕现。此可谓因陀罗网，交光互入，非复旧日蠹简之单行矣。

然庄生有言：“筌者所以在鱼，得鱼而忘筌；蹄者所以在兔，得兔而忘蹄。”今双链图谱，亦筌蹄之属也。点链纷繁，岂为炫目哉？将以通吾思也。苟能因链见义，由图得理，则链可忘，图可弃，所存者唯贯融之知见耳。若逐线而迷真，恋图而失旨，是犹执筌为鱼，刻舟求剑，去道愈远矣。

故君子用此器也，当以链为舟，以图为筏，既济则舍，不滞于物。笔记虽工，不过心影；图谱虽精，终属迹相。得意而忘器，则双链不必存，图谱不必设，而学自成矣。岂在区区之连线耶？
