# 仓库（Vault）

Tydora 使用"仓库（Vault）"概念管理笔记文件。每个仓库对应一个本地文件夹。

## 添加仓库

1. 点击侧栏底部的仓库切换器
2. 选择"添加仓库"
3. 在系统对话框中选择文件夹
4. 仓库将显示在侧栏文件树中

## 切换仓库

1. 点击侧栏底部的仓库切换器
2. 在下拉菜单中选择目标仓库
3. 侧栏将切换到该仓库的文件树

## 移除仓库

1. 点击侧栏底部的仓库切换器
2. 在仓库列表中悬停目标仓库
3. 点击删除图标
4. 确认删除

> 移除仓库不会删除实际文件夹，仅从 Tydora 中移除引用。

## 多仓库管理

Tydora 支持同时管理多个仓库，方便你按项目或主题组织笔记。

## 最近打开的文件

每个仓库会记录最近打开的文件，方便快速访问。

## 文件系统监控

Tydora 会监控仓库文件夹的外部修改，自动更新文件树。
