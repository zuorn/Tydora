# 编辑模式

Tydora 支持三种编辑模式，满足不同的编辑需求。

## 三种模式

### 所见即所得（WYSIWYG）

所见即所得模式下，你看到的就是最终输出的效果。文本格式化（如加粗、斜体）会立即显示，无需切换到预览。

**特点：**
- 直接显示格式化效果
- 适合熟悉 Markdown 的用户
- 支持拖拽调整图片大小

### 即时渲染（IR）

即时渲染模式在你输入 Markdown 语法后立即渲染结果，但保留部分语法标记供编辑。

**特点：**
- 输入时实时预览
- 保留语法上下文
- 适合边写边看效果

### 源码模式（SV）

源码模式显示纯 Markdown 文本，不进行任何渲染。

**特点：**
- 显示原始 Markdown 语法
- 适合精细调整格式
- 代码块内显示完整内容

## 切换模式

1. 按 `Ctrl+Shift+E` 循环切换模式
2. 或在 [[设置/编辑器设置]] 中设置默认编辑模式

## 模式对比

| 特性 | WYSIWYG | IR | SV |
|------|---------|-----|-----|
| 格式化显示 | 立即 | 延迟 | 无 |
| 语法可见性 | 低 | 中 | 高 |
| 适合场景 | 快速写作 | 边写边看 | 精细调整 |
