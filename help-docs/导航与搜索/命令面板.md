# 命令面板

命令面板让你快速搜索并执行 Tydora 的所有命令。

## 使用方法

1. 按 `Ctrl+P` 打开命令面板
2. 输入命令关键词进行模糊搜索
3. 从搜索结果中选择命令
4. 按 `Enter` 执行命令

## 支持的命令

### 编辑命令

- 撤销、重做
- 剪切、复制、粘贴
- 全选

### 格式命令

- 加粗、斜体、删除线
- 标题级别（1-6）
- 代码块、行内代码
- 列表、任务列表

### 视图命令

- 切换编辑模式
- 切换侧栏显示
- 打开思维导图
- 打开知识图谱

### 文件命令

- 新建文件
- 保存文件
- 关闭文件

## 搜索范围

命令面板搜索所有可用命令，包括：

- 菜单命令
- 快捷键命令
- 插件命令

## 最近命令

命令面板会记录最近使用的命令，方便快速重复执行。
