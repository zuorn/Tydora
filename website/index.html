核心功能

为热爱文字的人打造，每一项都围绕流畅写作体验。

三模式编辑器

基于 Vditor，支持所见即所得、即时渲染、分屏预览三种模式，所思即所写。

仓库管理

类 Obsidian 的多文件夹仓库，文件树浏览、新建重命名拖拽，一切井然有序。

命令面板

Ctrl+P 唤出，模糊搜索所有命令，最近使用智能排序，所想即可达。

快速打开

Ctrl+O 即可在仓库内模糊搜索任何文件，跳转只在弹指之间。

思维导图

Ctrl+M 基于 markmap，独立窗口自由缩放层级，让结构化思考一目了然。

多媒体预览

内置图片缩放、视频、音频、PDF 预览，无须切换应用，专注不受打断。

多主题系统

5 款精心打磨的内置主题，并兼容导入 Typora 主题文件，所见即所爱。

自动更新

基于 GitHub Releases 自动检测与安装，永远用上最新版本。

自定义快捷键

所有常用操作都可通过快捷键触发，按你的节奏工作。

大纲与统计

实时大纲面板跳转，字数统计、打字机模式，让长文写作井然有序。

数学公式

支持 KaTeX 与 MathJax 双引擎，学术写作同样优雅。

代码高亮

多款高亮主题、行号显示，为开发者而生的写作体验。

常见问题

关于 Tydora 你可能想知道的事。

Tydora 是免费的吗？

完全免费，且基于 MIT 协议开源。任何人都可以自由使用、修改并分发。

Tydora 与 Typora 是什么关系？

Tydora 是一个向 Typora 致敬的独立开源项目，并非官方分支或克隆。命名上保留了 "Ty" 与 "ora"，延续"指尖上的礼物"这一浪漫期许。

Tydora 支持哪些平台？

当前主要面向 Windows（NSIS 安装包）；macOS 构建正在筹备中，敬请期待。

可以导入 Typora 主题吗？

可以。Tydora 兼容 Typora 的主题 CSS 文件格式，在应用设置中导入即可生效。

如何参与贡献？

欢迎在 GitHub 上提交 Issue 报告问题，或 Fork 后发起 Pull Request。每一份贡献都让 Tydora 走得更远。
