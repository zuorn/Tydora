# 内置主题

Tydora 提供 8 个精心设计的内置主题。

## 主题列表

### Catppuccin Mocha

柔和的深色主题，以粉色和棕色为主色调，适合长时间写作。

### 白色（White）

简洁的浅色主题，纯净的白色背景，适合日常使用。

### Mint

清新的薄荷绿色主题，以绿色和白色为主，清新自然。

### Mint Dark

Mint 的深色版本，适合喜欢绿色调但偏好深色的用户。

### Liquid Glass

玻璃质感主题，半透明效果，现代感十足。

### Claude Code

灵感来自 Claude 代码编辑器的深色主题。

### Purple

优雅的紫色主题，适合喜欢紫色调的用户。

### Hermes

温暖的橙色主题，活力十足。

## 切换主题

### 方法一：设置面板

1. 打开设置（`Ctrl+,`）
2. 切换到"主题"标签
3. 点击选择主题

### 方法二：仓库切换器

1. 点击侧栏底部的仓库切换器
2. 点击设置图标
3. 选择主题

## 外观模式

主题支持三种外观模式：

- **跟随系统** — 自动切换深色/浅色
- **浅色** — 强制浅色模式
- **深色** — 强制深色模式

可在 [[设置/通用设置]] 中配置。
