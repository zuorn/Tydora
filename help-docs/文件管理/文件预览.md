# 文件预览

Tydora 支持多种非 Markdown 文件的预览。

## 支持的格式

### 图片

JPG、PNG、GIF、WebP、BMP、SVG、AVIF、HEIC、ICO

### 视频

MP4、WebM、OGG、MOV、AVI、MKV、WMV、FLV、M4V

### 音频

MP3、WAV、OGG、FLAC、AAC、M4A、WMA

### 文档

PDF

## 预览方式

在侧栏文件树中双击文件，或在编辑器中点击图片链接，即可打开预览。

## 图片预览

- 支持缩放和拖拽
- 显示图片尺寸信息
- 支持透明背景显示

## 视频预览

- 内置播放控件
- 支持全屏播放
- 显示进度条

## 音频预览

- 内置播放控件
- 显示播放进度

## PDF 预览

- 内置 PDF 查看器
- 支持翻页和缩放
