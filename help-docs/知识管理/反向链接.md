# 反向链接

反向链接面板显示哪些笔记链接到了当前笔记，帮你发现笔记间的关联。

## 功能说明

- 实时显示引用当前笔记的所有来源
- 显示引用上下文片段
- 支持点击跳转到来源笔记

## 使用方法

1. 打开一个笔记
2. 在侧栏顶部切换到"反向链接"标签
3. 查看哪些笔记引用了当前笔记
4. 点击任意条目跳转到来源笔记

## 使用场景

- 发现笔记间的隐藏关联
- 回顾哪些内容引用了当前笔记
- 评估笔记的重要性（被引用越多越重要）
- 检查断链或孤立笔记

## 链接索引

反向链接功能依赖 [[链接索引]] 服务。Tydora 会在后台自动构建和维护链接索引。
