# Typora 主题

Tydora 支持导入和使用 Typora 的 CSS 主题文件。

## 导入方法

1. 打开设置（`Ctrl+,`）
2. 切换到"主题"标签
3. 点击"导入主题"
4. 选择 Typora 的 CSS 主题文件
5. 主题将添加到主题列表中

## 主题文件位置

导入的主题会保存在 Tydora 的主题目录中。

可在 [[设置/通用设置]] 中查看和修改主题资源目录。

## 兼容性说明

Tydora 基于 Vditor 引擎，与 Typora 的渲染引擎不同，部分主题效果可能有差异。

### 兼容的部分

- 颜色方案
- 字体设置
- 基础布局
- 代码块样式

### 可能有差异的部分

- 复杂的 CSS 动画
- 特定的 DOM 结构
- 某些自定义组件样式

## 管理主题

### 查看已导入主题

在设置面板的"主题"标签中查看所有可用主题。

### 删除主题

1. 在主题列表中悬停目标主题
2. 点击删除图标
3. 确认删除
