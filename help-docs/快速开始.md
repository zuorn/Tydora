# 快速开始

Tydora 是一款基于 Tauri 构建的现代化桌面 Markdown 编辑器，支持 WYSIWYG 编辑、知识管理、思维导图等功能。

## 安装

1. 从 [GitHub Releases](https://github.com/tydora/tydora/releases) 下载最新版本
2. 运行安装程序，按照提示完成安装
3. 启动 Tydora

## 打开仓库

Tydora 使用"仓库（Vault）"概念管理笔记文件。首次使用时：

1. 点击侧栏底部的仓库切换器
2. 选择"添加仓库"
3. 选择一个文件夹作为你的笔记仓库
4. 侧栏将显示该文件夹的文件树

> 参见 [[仓库]] 了解更多仓库管理功能。

## 创建笔记

1. 在侧栏文件树中右键点击
2. 选择"新建文件"
3. 输入文件名（如 `我的第一篇笔记.md`）
4. 开始编写 Markdown 内容

## 基本编辑

Tydora 支持三种 [[编辑模式]]：

- **所见即所得（WYSIWYG）**：直接在编辑器中看到最终效果
- **即时渲染（IR）**：输入 Markdown 语法后立即渲染
- **源码模式（SV）**：纯文本编辑，显示原始 Markdown

按 `Ctrl+Shift+E` 切换编辑模式。

## 保存文件

- 按 `Ctrl+S` 保存当前文件
- 启用自动保存后，文件会在修改后自动保存（可在 [[通用设置]] 中配置）

## 快速导航

- `Ctrl+O` — [[快速打开]] 文件
- `Ctrl+P` — [[命令面板]] 搜索命令
- `Ctrl+\` — 切换侧栏显示/隐藏
