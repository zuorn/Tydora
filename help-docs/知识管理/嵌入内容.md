# 嵌入内容

嵌入功能让你在当前笔记中显示其他笔记的内容。

## 基本语法

```markdown
![[笔记名]]
```

## 嵌入标题

嵌入其他笔记的特定标题及其子内容：

```markdown
![[笔记名#标题]]
```

## 嵌入图片

嵌入仓库中的图片文件：

```markdown
![[图片路径.png]]
```

## 嵌入视频/音频

嵌入仓库中的视频或音频文件：

```markdown
![[视频路径.mp4]]
![[音频路径.mp3]]
```

## 使用场景

- 引用其他笔记的段落
- 集中管理图片资源
- 创建内容聚合页面
- 复用常用模板

## 注意事项

- 嵌入的内容是只读的，不能在当前笔记中编辑
- 修改源笔记会影响所有嵌入该内容的地方
- 支持嵌入图片、视频、音频等多媒体文件
