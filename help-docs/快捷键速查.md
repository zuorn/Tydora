# 快捷键速查表

Tydora 支持 40+ 个键盘快捷键，以下为默认配置。

## 格式

| 快捷键 | 功能 |
|--------|------|
| `Ctrl+B` | 加粗 |
| `Ctrl+I` | 斜体 |
| `Ctrl+D` | 删除线 |
| `Ctrl+E` | 行内代码 |
| `Ctrl+K` | 插入链接 |
| `Ctrl+Shift+I` | 插入图片 |

## 列表

| 快捷键 | 功能 |
|--------|------|
| `Ctrl+Shift+U` | 无序列表 |
| `Ctrl+Shift+O` | 有序列表 |
| `Ctrl+Shift+X` | 任务列表 |

## 标题

| 快捷键 | 功能 |
|--------|------|
| `Ctrl+1` | 一级标题 |
| `Ctrl+2` | 二级标题 |
| `Ctrl+3` | 三级标题 |
| `Ctrl+4` | 四级标题 |
| `Ctrl+5` | 五级标题 |
| `Ctrl+6` | 六级标题 |
| `Ctrl+0` | 段落 |

## 插入

| 快捷键 | 功能 |
|--------|------|
| `Ctrl+Shift+T` | 插入表格 |
| `Ctrl+Shift+K` | 代码块 |

## 编辑

| 快捷键 | 功能 |
|--------|------|
| `Ctrl+Z` | 撤销 |
| `Ctrl+Y` | 重做 |
| `Ctrl+X` | 剪切 |
| `Ctrl+C` | 复制 |
| `Ctrl+V` | 粘贴 |
| `Ctrl+A` | 全选 |
| `Tab` | 增加缩进 |
| `Shift+Tab` | 减少缩进 |

## 视图

| 快捷键 | 功能 |
|--------|------|
| `Ctrl+\` | 切换侧栏 |
| `Ctrl+Shift+E` | 切换编辑模式 |
| `Ctrl+Shift+T` | 切换打字机模式 |
| `Ctrl+M` | 打开思维导图 |
| `Ctrl+G` | 打开知识图谱 |

## 系统

| 快捷键 | 功能 |
|--------|------|
| `Ctrl+S` | 保存文件 |
| `Ctrl+O` | 快速打开 |
| `Ctrl+P` | 命令面板 |
| `Ctrl+,` | 打开设置 |
| `F2` | 重命名 |
| `Delete` | 删除 |

## 自定义快捷键

所有快捷键都可以在 [[设置/快捷键设置]] 中自定义。
