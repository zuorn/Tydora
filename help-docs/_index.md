# Tydora 帮助文档

欢迎使用 Tydora 帮助文档。Tydora 是一款基于 Tauri 构建的现代化桌面 Markdown 编辑器，支持知识管理、双链笔记、思维导图等功能。

## 快速开始

- [[快速开始]] — 首次使用指南

## 编辑器

- [[编辑模式]] — WYSIWYG、即时渲染、源码三种编辑模式
- [[右键菜单]] — 格式化命令：标题、加粗、斜体、代码、引用、列表等
- [[打字机模式]] — 光标自动居中，专注写作
- [[Markdown语法]] — 支持的语法列表
- [[表格操作]] — 表格编辑快捷键
- [[数学公式]] — KaTeX / MathJax 公式支持
- [[代码块]] — 代码高亮、主题、行号

## 文件管理

- [[仓库]] — 多仓库系统，管理多个笔记目录
- [[文件树]] — 递归目录、展开折叠、拖拽排序
- [[文件操作]] — 新建、重命名、删除、拖拽移动、复制路径
- [[文件预览]] — 图片、视频、音频、PDF 预览

## 导航与搜索
