# 快捷键设置

Tydora 支持 40+ 个可自定义的键盘快捷键。

## 打开设置

按 `Ctrl+,` 打开设置面板，切换到"快捷键"标签。

## 快捷键分组

### 格式

加粗、斜体、删除线、行内代码、链接等。

### 列表

无序列表、有序列表、任务列表。

### 标题

1-6 级标题切换。

### 插入

表格、代码块、链接、图片等。

### 表格

表格编辑相关快捷键。

### 编辑

撤销、重做、剪切、复制、粘贴。

### 视图

侧栏切换、编辑模式切换、思维导图等。

### 系统

保存、打开设置、打开命令面板等。

## 自定义快捷键

1. 在快捷键列表中找到目标命令
2. 点击当前快捷键
3. 按下新的快捷键组合
4. 保存

## 重置快捷键

点击"重置"按钮可将所有快捷键恢复为默认值。

## 冲突检测

如果新设置的快捷键与现有快捷键冲突，系统会提示确认。
