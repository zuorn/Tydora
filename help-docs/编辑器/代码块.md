# 代码块

Tydora 支持代码语法高亮，支持 36 种编程语言。

## 创建代码块

### 方法一：Markdown 语法

使用三个反引号包裹代码，指定语言名称：

````markdown
```javascript
function hello() {
    console.log("Hello, Tydora!");
}
```
````

### 方法二：右键菜单

1. 右键点击编辑区域
2. 选择"插入" → "代码块"
3. 选择编程语言

## 支持的语言

JavaScript、TypeScript、Python、Java、C/C++、Go、Rust、Ruby、PHP、Swift、Kotlin、SQL、HTML、CSS、Shell、Markdown 等 36 种语言。

## 行内代码

使用单个反引号包裹代码：

```markdown
使用 `console.log()` 输出日志
```

## 代码高亮主题

Tydora 支持多种代码高亮主题，可在 [[设置/编辑器设置]] 中切换：

- Atom One Light / Dark
- GitHub Light / Dark
- VS Code Light / Dark
- Nord
- Monokai
- Dracula
- Solarized

> 详见 [[代码高亮主题]]

## 行号

代码块默认显示行号。可在 [[设置/编辑器设置]] 中关闭。

## 复制代码

点击代码块右上角的复制按钮，可快速复制代码内容。
