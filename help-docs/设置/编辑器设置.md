# 编辑器设置

编辑器设置管理编辑行为、渲染选项、语法高亮等配置。

## 打开设置

按 `Ctrl+,` 打开设置面板，切换到"编辑器"标签。

## 编辑模式

设置默认的 [[编辑模式]]：

- 所见即所得（WYSIWYG）
- 即时渲染（IR）
- 源码模式（SV）

## 打字机模式

启用后，光标始终保持在屏幕中央。

> 详见 [[打字机模式]]

## Markdown 渲染

### 数学公式

启用/禁用数学公式渲染，选择公式引擎（KaTeX / MathJax）。

> 详见 [[数学公式]]

### 代码高亮

启用/禁用代码语法高亮，选择高亮主题。

> 详见 [[代码高亮主题]]

### Mermaid 图表

启用/禁用 Mermaid 图表渲染。

### 脚注

启用/禁用脚注语法。

### 目录

启用/禁用 `[toc]` 目录语法。

## 字数统计

显示当前文档的字数、字符数、行数。

## 缓存设置

清除编辑器缓存，解决显示问题。

## XSS 过滤

启用后，过滤 Markdown 中的潜在 XSS 攻击代码。
