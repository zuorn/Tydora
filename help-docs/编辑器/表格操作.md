# 表格操作

Tydora 提供便捷的表格编辑功能。

## 创建表格

### 方法一：Markdown 语法

```markdown
| 列1 | 列2 | 列3 |
|-----|-----|-----|
| 内容 | 内容 | 内容 |
```

### 方法二：右键菜单

1. 右键点击编辑区域
2. 选择"插入" → "表格"
3. 选择行列数

### 方法三：快捷键

按 `Ctrl+Shift+T` 快速插入表格

## 编辑表格

### 添加行列

- 将光标置于表格最后一列，按 `Tab` 添加新行
- 将光标置于表格最后一行，按 `Enter` 添加新行

### 删除行列

- 选中要删除的行或列
- 右键选择"删除行"或"删除列"

### 对齐方式

在表格分隔行中使用冒号设置对齐：

```markdown
| 左对齐 | 居中对齐 | 右对齐 |
|:-------|:-------:|-------:|
| 内容 | 内容 | 内容 |
```

## 快捷键

| 快捷键 | 功能 |
|--------|------|
| `Tab` | 移动到下一列 / 添加新行 |
| `Shift+Tab` | 移动到上一列 |
| `Enter` | 添加新行 |
