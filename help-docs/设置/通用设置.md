# 通用设置

通用设置管理应用的外观、字体、自动保存等基础选项。

## 打开设置

按 `Ctrl+,` 打开设置面板，默认显示"通用"标签。

## 外观模式

设置应用的深色/浅色模式：

- **跟随系统** — 自动匹配系统主题
- **浅色** — 强制浅色模式
- **深色** — 强制深色模式

## 字体设置

### 编辑器字体

选择编辑器中使用的字体：

- 系统默认
- LXGW WenKai（霞鹜文楷）
- 等宽字体
- 其他系统已安装字体

### 字号

调整编辑器文本大小，范围 12-32px。

## 自动保存

启用后，文件在修改后自动保存。

### 延迟时间

设置自动保存的延迟时间（秒），避免频繁保存。

## 主题资源目录

查看和修改主题文件的存储位置。导入的 [[Typora主题]] 会保存在此目录。
